# AppMagic Crawler

Extract top charts data from [AppMagic](https://appmagic.rocks/top-charts/apps) — no server, no login.

## Quick start

1. Open **[appmagic_bookmarklet.html](appmagic_bookmarklet.html)** and drag the bookmarklet to your bookmarks bar
2. Go to [appmagic.rocks/top-charts/apps](https://appmagic.rocks/top-charts/apps)
3. Click the bookmarklet → data extracts, download JSON or CSV
