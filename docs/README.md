# AppMagic Crawler

Paste an AppMagic link → crawl data → export JSON/CSV.

## Deploy

[Deploy to Render](https://render.com/deploy?repo=https://github.com/lamtrang2405/crawl-data) (free) → paste AppMagic URL, click Crawl.

## Live

**https://lamtrang2405.github.io/crawl-data/** — enter your Render URL when prompted.
